# Customer Analysis Dashboard Project

This project presents an end-to-end data analytics workflow focused on customer behavior analysis and business insight generation.

The process began in **Jupyter Notebook**, where Python (Pandas, NumPy, Matplotlib) was used for data cleaning, preprocessing, and exploratory data analysis (EDA). Missing values, inconsistent formats, and outliers were handled to ensure data quality. Key patterns such as purchasing behavior, seasonal trends, and customer segments were identified through visualizations.

The cleaned dataset was then stored and managed in **PostgreSQL**, where structured SQL queries were used to extract meaningful insights. Queries were written to compute KPIs such as total revenue, average order value, repeat purchase rate, and product/category performance.

Next, the processed data was connected to **Power BI**, where an interactive dashboard was built. The dashboard includes dynamic filters, KPI cards, and visualizations that clearly communicate trends, customer segmentation, and business performance metrics.

Finally, insights from the dashboard were compiled into a professional presentation (PPT) designed for stakeholders/clients. The presentation highlights objectives, methodology, key findings, and actionable recommendations to support data-driven decision-making.

**Tech Stack:** Python (Pandas, NumPy, Matplotlib), PostgreSQL (SQL), Power BI
**Outcome:** A complete analytics pipeline from raw data to business-ready insights and presentation.
